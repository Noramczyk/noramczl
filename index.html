Web-based Recursive Descent Recognizer



This web based recognizer tests strings that are valid within the grammar below.

EXP ::= EXP + TERM | EXP - TERM | TERM
TERM ::= TERM * FACTOR | TERM / FACTOR | FACTOR
FACTOR ::= ( EXP ) | DIGIT
DIGIT ::= 0 | 1 | 2 | 3


To use this Recognizer simply enter in a string and follow it with a '$'. Then press enter....
For example: (1+3)$ is valid and (1+3) is not.
